# BPDACC Icons Directory

## Folder Structure
```
icons/
├── sidebar/          # Sidebar navigation icons & app logo
├── dashboard/        # Dashboard stat & activity icons
├── inventory/        # Inventory action icons
├── reports/          # Reports stat icons
├── users/            # Users page action icons
└── settings/         # Settings page action icons
```

---

## 💡 Pro Tip

- Clean, modern, minimalist style
- Perfect for a medical/clinic application
- All icons are consistent and match each other
- Free to use!
